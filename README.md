# Student Performance Analysis

This project analyzes student academic performance using Python and Pandas.

## What I Did
- Cleaned and handled missing values in student data
- Created performance, attendance-risk, and improvement indicators
- Analyzed the effect of attendance and study habits on grades
- Visualized key insights using Matplotlib

## Tools Used
Python, Pandas, NumPy, Matplotlib

## Key Insight
Students with lower attendance risk show a higher proportion of high academic performers.
